\doxysubsection{Подробное описание}
Класс \doxylink{class_sets}{Sets} предоставляет функциональность для управления наборами упражнений (сетами). 

Этот класс позволяет загружать, добавлять, удалять и отображать наборы упражнений, а также устанавливать соединения между элементами пользовательского интерфейса.
