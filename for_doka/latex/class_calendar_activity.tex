\doxysubsection{Подробное описание}
Класс для управления активностью календаря в приложении. 

\doxylink{class_calendar_activity}{Calendar\+Activity} представляет собой диалоговое окно, отображающее календарь и список упражнений пользователя, а также позволяет управлять этими данными.
