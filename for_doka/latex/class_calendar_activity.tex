\doxysubsection{Detailed Description}
Класс \doxylink{class_calendar_activity}{Calendar\+Activity} представляет собой диалоговое окно для управления активностями календаря. 

Класс \doxylink{class_calendar_activity}{Calendar\+Activity} управляет добавлением, удалением и отображением активностей и сетов упражнений в календаре.
